# Платформа мини-игр в Telegram

Telegram-бот с двумя мини-играми, системой рейтинга и панелью администратора.  
## Игры
- **Угадай число** - попробуй угадать число
- **Викторина** - проверь свою эрудицию
---

## Стэк технологий

- Python 3.13
- telegram-bot-api (через `requests`)
- SQLite (встроенный модуль `sqlite3`)
- `pytest` 

---


## Как запустить проект

### 1. **Скопируйте репозиторий**
```bash
git clone <link>
cd tg_games
```
### 2. **Установите зависимости:**
```
pip install -r requirements.txt
```
### 3. **Создайте файл `.env`:**
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
```
Токен можно получить в телеграмме у `@BotFather`, ID администратора можно получить у `@userinfobot`.
### 4. **Запуск проекта**
```
python run.py
```
### 5. **Запуск тестов**
```
pytest tests/
```
### 6. **Запуск бота** (опционально)
Повторите шаг 4 и напишите боту /start - готово.
